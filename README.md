# Neovim Configs
Neovim configuration files for different plugin managers

# Neovim (Unix\Linux)
### Installation
```
curl -LO https://github.com/neovim/neovim/releases/latest/download/nvim.appimage
chmod u+x nvim.appimage
./nvim.appimage
```

### Expose Globally
```
mv nvim.appimage /usr/bin/nvim
```
